# 概要
2019年3月13日(水曜日)に開催した、LINEが提供する法人向けサービスに関するパートナーセミナーに使用したLINE BOTとClovaスキルのソースコードです。
# 内容
| ソース                          | 説明                                                 |
|:-------------------------------|:----------------------------------------------------|
| [line-bot](./line-bot)         |セミナー出欠確認、アンケート回答収集、質問受付を実現するLINE BOT  |
| [clova-skill](./clova-skill)   |受けた質問を読み上げるClovaスキル                          |
